# 远洋渔业智能驾驶舱系统（dd-cockpit）

## 项目概述

远洋渔业智能驾驶舱系统是一个面向舟山远洋渔业的综合性管理平台，通过集成 GIS 地图、实时数据监控、AI 智能分析等技术，为渔业管理部门提供船舶监控、天气预警、渔获管理等一站式解决方案，助力渔业数字化转型和智能化管理。

## 技术栈

### 核心框架
- **前端框架**: Vue 3 + TypeScript + Vite
- **UI 框架**: Ant Design Vue + Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router

### 地图与可视化
- **地图引擎**: OpenLayers + Cesium + 高德地图
- **数据可视化**: ECharts + ECharts GL + ECharts WordCloud
- **3D 渲染**: Three.js

### 实时通信与数据处理
- **实时通信**: Socket.IO Client
- **HTTP 请求**: Axios
- **数据压缩**: pako
- **加密**: crypto-js

### 视频与多媒体
- **视频播放**: Video.js + HLS.js
- **视频监控**: web-control

### 工具库
- **工具函数**: lodash-es
- **日期处理**: dayjs + moment
- **DOM 操作**: D3.js
- **轮播组件**: Swiper
- **文档生成**: html2canvas + jspdf
- **Vue 工具**: VueUse

### 开发工具
- **代码规范**: ESLint + Prettier
- **包管理**: pnpm
- **构建工具**: Vite + Rollup

## 核心功能模块

### 1. 地图服务
- 多种地图切换：海图、卫星图、三维地图
- 电子围栏管理
- 图层控制与图例展示
- 地图工具栏

### 2. 船舶监控
- 实时船舶位置监控
- 船舶轨迹回放
- 船舶详情查看
- 船舶搜索与筛选

### 3. 天气服务
- 台风路径监控与预警
- 水位监测
- 波高、水温监测
- 气压、降雨监测
- 潮汐预报
- 能见度监测

### 4. 预警系统
- 人员预警
- 船舶预警
- 预警详情弹窗
- 预警列表管理

### 5. 渔获贸易
- 渔获商品管理
- 渔获贸易情况
- 渔区管理
- 渔场管理

### 6. AI 智能助手
- AI 大模型对话功能
- Markdown 文本解析
- 智能问答与建议
- 响应内容格式化显示

### 7. 视频监控
- 实时视频监控
- 视频列表管理
- 视频播放器组件
- HLS 视频流支持

### 8. 搜索功能
- 船舶搜索（支持 MMSI、船名等）
- 船员搜索
- Tabs 切换功能
- 搜索结果映射与显示
