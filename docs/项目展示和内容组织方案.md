## 🖼️ 项目截图准备

### 截图规范

#### 1. 基本要求
- **分辨率**: 1920x1080 或更高
- **格式**: PNG 或 WebP
- **文件大小**: < 500KB(压缩后)
- **命名规范**: `{project-id}-{scene}-{index}.jpg`

#### 2. 截图场景

每个项目建议准备以下截图:

##### 政企项目
```
1. 主界面/监控大屏
   - 展示整体界面布局
   - 突出核心功能区域

2. 地图监控界面
   - 展示地图和船舶位置
   - 突出GIS功能

3. 预警面板
   - 展示预警列表和详情
   - 突出预警功能

4. 视频监控界面
   - 展示视频窗口
   - 突出视频功能

5. 数据统计图表
   - 展示数据可视化
   - 突出分析功能
```

##### AI项目
```
1. 对话界面
   - 展示聊天界面
   - 突出对话功能

2. 模型设置
   - 展示模型配置
   - 突出模型支持

3. 插件中心
   - 展示插件列表
   - 突出扩展能力

4. 工作流设计
   - 展示工作流编排
   - 突出逻辑设计
```

#### 3. 截图处理

```bash
# 使用ImageMagick压缩图片
convert input.png -quality 85 -strip output.jpg

# 批量处理
for file in *.png; do
  convert "$file" -quality 85 -strip "${file%.png}.jpg"
done
```

#### 4. 截图标注(可选)

对于复杂界面,可以添加标注突出关键功能:

```
1. 使用工具:
   - Skitch
   - Mark Text
   - Photoshop

2. 标注内容:
   - 核心功能区域
   - 关键数据指标
   - 重要交互元素

3. 标注样式:
   - 统一的颜色方案
   - 清晰的字体
   - 适度的标注数量(不超过5个)
```

```vue
<script setup lang="ts">
import { ref, computed } from 'vue';
import { useVirtualList } from '@vueuse/core';

const projects = ref<Project[]>([]);
const pageSize = 9;
const currentPage = ref(1);

const paginatedProjects = computed(() => {
  const start = (currentPage.value - 1) * pageSize;
  const end = start + pageSize;
  return projects.value.slice(start, end);
});

// 或使用虚拟滚动
const { list, containerProps, wrapperProps } = useVirtualList(
  projects,
  { itemHeight: 400 }
);
</script>
```

### 4. 懒加载

```vue
<template>
  <div v-if="isVisible" class="project-card">
    <!-- 项目内容 -->
  </div>
</template>

<script setup lang="ts">
import { ref, onMounted } from 'vue';

const isVisible = ref(false);

onMounted(() => {
  const observer = new IntersectionObserver(
    ([entry]) => {
      if (entry.isIntersecting) {
        isVisible.value = true;
        observer.disconnect();
      }
    },
    { threshold: 0.1 }
  );

  observer.observe(el.value);
});
</script>
```
