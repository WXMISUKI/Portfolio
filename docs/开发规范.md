## 🎯 项目概述

### 项目定位
本网站是一个面向求职展示的个人作品集网站,展示前端工程和AI全栈开发能力。主要目标是:
- 向HR和技术面试官展示个人技术实力
- 突出政企项目和AI智能体项目的实战经验
- 体现工程化思维和最佳实践
- 提供优秀的用户体验和视觉设计

### 技术栈

#### 核心技术
- **前端框架**: Vue 3.5+ (Composition API)
- **开发语言**: TypeScript 5.0+
- **构建工具**: Vite 6.0+
- **UI框架**: 原生组件 + UnoCSS
- **状态管理**: Pinia 3.0+
- **路由**: Vue Router 4.5+

#### 样式方案
- **原子化CSS**: UnoCSS 0.60+
- **动画库**: VueUse + CSS Transitions
- **图标**: UnoCSS Preset Icons

#### 工具库
- **HTTP客户端**: Axios
- **工具函数**: Lodash-es
- **日期处理**: Day.js
- **滚动交互**: VueUse (useScroll, useElementVisibility)

#### 开发工具
- **代码规范**: ESLint + Prettier
- **Git Hooks**: Husky + lint-staged
- **包管理**: pnpm

### 设计风格
- **主题**: 深色主题、科技风、极简
- **配色**: 深蓝主色(#0F172A)、青色强调色(#06B6D4)、深色背景(#020617)、高可读灰白文字(#E2E8F0)

- **交互**: hover动效、滚动渐入、平滑过渡、移动端完美适配

import { ref, onMounted, onUnmounted } from 'vue';

export function useScrollReveal() {
  // 1. 响应式状态
  const isVisible = ref(false);
  const threshold = ref(0.1);

  // 2. 内部方法
  const handleScroll = () => {
    // 滚动逻辑
  };

  // 3. 生命周期
  onMounted(() => {
    window.addEventListener('scroll', handleScroll);
  });

  onUnmounted(() => {
    window.removeEventListener('scroll', handleScroll);
  });

  // 4. 返回公共API
  return {
    isVisible,
    threshold
  };
}

## 🧩 组件开发规范

### 组件设计原则
1. **单一职责**: 每个组件只做一件事
2. **可复用性**: 通过props实现组件的可配置性
3. **可测试性**: 组件逻辑清晰,易于测试
4. **性能优化**: 避免不必要的重渲染

### Props规范
```typescript
interface Props {
  // 必需属性
  id: string;
  title: string;

  // 可选属性使用默认值
  description?: string;
  image?: string;

  // 枚举类型
  size?: 'sm' | 'md' | 'lg';

  // 布尔值
  disabled?: boolean;
  isVisible?: boolean;

  // 数组/对象
  tags?: string[];
  metadata?: Record<string, any>;
}

const props = withDefaults(defineProps<Props>(), {
  description: '',
  image: '',
  size: 'md',
  disabled: false,
  isVisible: true,
  tags: () => [],
  metadata: () => ({}),
});
```

// 定义明确的事件类型

## 🔄 状态管理规范

### Pinia Store规范
```typescript
// store/modules.ts
import { defineStore } from 'pinia';

interface ProjectState {
  projects: Project[];
  currentProject: Project | null;
  isLoading: boolean;
  error: string | null;
}

export const useProjectStore = defineStore('project', {
  state: (): ProjectState => ({
    projects: [],
    currentProject: null,
    isLoading: false,
    error: null,
  }),

  getters: {
    // 获取活跃项目
    activeProjects: (state) => state.projects.filter(p => p.active),

    // 按类型分组
    projectsByType: (state) => {
      return state.projects.reduce((acc, project) => {
        if (!acc[project.type]) {
          acc[project.type] = [];
        }
        acc[project.type].push(project);
        return acc;
      }, {} as Record<string, Project[]>);
    },
  },

  actions: {
    // 加载项目列表
    async loadProjects() {
      this.isLoading = true;
      this.error = null;
      try {
        const data = await fetchProjects();
        this.projects = data;
      } catch (error) {
        this.error = '加载失败';
        console.error('Failed to load projects:', error);
      } finally {
        this.isLoading = false;
      }
    },

    // 设置当前项目
    setCurrentProject(project: Project) {
      this.currentProject = project;
    },
  },
});
```

// composables/usePerformance.ts
export function usePerformance() {
  const measureRender = (name: string, fn: () => void) => {
    performance.mark(`${name}-start`);
    fn();
    performance.mark(`${name}-end`);
    performance.measure(name, `${name}-start`, `${name}-end`);

    const measure = performance.getEntriesByName(name)[0];
    console.log(`${name} 耗时: ${measure.duration.toFixed(2)}ms`);

    // 清理性能标记
    performance.clearMarks();
    performance.clearMeasures();
  };

  return {
    measureRender,
  };
}
