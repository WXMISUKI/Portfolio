# 嘉兴公安海防反走私系统

## 项目概述

**项目名称**：嘉兴公安海防反走私系统  
**项目类型**：政府信息化系统 / 地理信息系统（GIS）  
**开发周期**：2025年10月 - 至今  
**项目规模**：大型企业级应用，支持多用户并发访问  
**项目角色**：前端开发工程师  
**项目描述**：

嘉兴公安海防反走私系统是一个基于Web的综合性海防监控系统，集成了实时船舶跟踪、电子围栏管理、预警规则配置、技防评估等核心功能。系统通过OpenLayers地图引擎实现船舶实时监控和轨迹回放，为公安部门提供全方位的海防反走私监管能力，有效提升海上安全监控效率和预警响应速度。

---

## 技术栈

### 核心技术
- **前端框架**：Vue 3.5（Composition API）
- **开发语言**：TypeScript
- **UI组件库**：Element Plus 2.9
- **构建工具**：Vite 6.3
- **状态管理**：Pinia + pinia-plugin-persistedstate
- **路由管理**：Vue Router 4.5

### 地图与可视化
- **地图引擎**：OpenLayers 9.2 + MapLibre GL 5.7
- **数据可视化**：ECharts 5.6
- **地理计算**：@turf/turf 7.2 + proj4 2.19

### 网络与数据
- **HTTP客户端**：Axios 1.8.4
- **数据加密**：crypto-js 4.2 + jsencrypt 3.3
- **数据缓存**：IndexedDB（本地存储优化）

### 工具库
- **国际化**：Vue I18n 11.1
- **工具函数**：@vueuse/core 13.9
- **文件处理**：file-saver 2.0.5
