# 舟山大桥综管指挥平台 - 简历项目说明

## 项目简介
舟山大桥综管指挥平台是一个面向大桥运营管理的综合性数字化指挥系统，实现设备监控、应急响应、巡检跟踪、结构监测、交通管理、视频监控等关键业务流程的数字化和可视化，显著提升大桥运营管理效率。

## 技术栈
**核心框架**：Vue3 + TypeScript + Vite + Element Plus + Pinia  
**可视化**：ECharts + OpenLayers（天地图）  
**视频播放**：hls.js + flv.js + Video.js  
**工具库**：@vueuse/core + dayjs + html2canvas + Axios  
**样式方案**：UnoCSS + Sass + Less  
**开发工具**：ESLint + Prettier + TypeScript

## 主要职责
作为前端开发工程师，负责核心功能模块的开发和优化，包括预警管理、应急指挥、视频监控、地图系统等关键模块。

## 核心功能模块

### 1. 预警管理系统
- **多维度预警门户**：实现交通事件、环境污染、地质灾害、公共卫生、气象灾害、船舶等6大类预警的统一管理
- **预警附件功能**：为所有预警类型添加附件上传、预览、下载功能，支持多种文件格式
- **预警研判转事件**：实现预警信息的智能研判和转事件功能，支持批量操作
- **实时预警统计**：实现各类预警未处理数量的实时统计和展示

### 2. 应急指挥系统
- **事件处置流程**：开发事件全流程管理功能，实现前期处置、应急响应、应急终止等节点管理
- **专项预案管理**：实现专项预案的创建、编辑、删除和流程集成
- **应急处置工作总结报告**：自动生成专业报告，支持数据自动收集、导出PNG和PDF，采用官方公文样式
- **事件类型映射**：完善事件类型映射关系，优化事件分类和统计功能

### 3. 视频监控系统
- **多格式视频播放**：集成hls.js、flv.js、Video.js，支持HLS、FLV、原生视频等多种格式
- **自定义控制栏**：实现播放/暂停、音量控制、全屏、倍速播放等功能
- **检测框绘制**：基于Canvas实现视频检测框的绘制和交互功能
- **多路视频监控**：实现多路视频的实时监控和回放功能

### 4. 地图与船舶监控
- **地图坐标选择器**：基于OpenLayers实现地图坐标选择功能，集成天地图服务
- **电子围栏系统**：实现圆形、矩形、多边形等多种类型的电子围栏绘制和管理
- **船舶实时监控**：实现船舶位置实时显示、航迹追踪、分组管理
- **警戒船对接**：实现警戒船的数据对接、实时监控和警戒范围管理
- **巡航功能**：开发船舶巡航功能，实现巡航路线规划和过程监控

### 5. 系统管理
- **权限管理**：基于RBAC模型，实现路由、按钮、数据多维度权限控制
- **用户管理**：实现用户、角色、部门的统一管理
- **SSO单点登录**：封装SSO单点登录组件，提升系统安全性

## 解决的技术难点

### 1. 复杂工作流管理
- **问题**：事件处置流程复杂，涉及多个节点和状态，需要动态渲染和实时更新
- **解决**：使用状态机模式管理流程状态，采用组件化设计，每个流程节点独立组件，使用Pinia进行集中式状态管理

### 2. 地图性能优化
- **问题**：大桥区域广，地图数据量大，需要实时显示大量船舶位置
- **解决**：使用OpenLayers的矢量图层优化渲染，实现地图数据的缓存机制，采用Web Worker处理复杂计算

### 3. 视频流播放
- **问题**：需要支持多种视频流格式，播放延迟低，需要支持检测框绘制
- **解决**：集成多种视频播放库，实现自定义视频控制栏，使用Canvas绘制检测框，优化视频流的缓冲策略

### 4. 实时数据更新
- **问题**：系统需要实时更新大量数据，需要保证数据的准确性和及时性
- **解决**：使用WebSocket实现实时通信，实现数据更新节流机制，使用虚拟列表优化大数据渲染
