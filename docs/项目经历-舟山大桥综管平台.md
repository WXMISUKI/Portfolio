# 项目经历 - 舟山跨海大桥综合监管平台

## 项目基本信息

- **项目名称**：舟山跨海大桥综合监管平台
- **项目角色**：前端开发工程师
- **开发周期**：2024年9月 - 2025年12月
- **技术栈**：Vue 3 + TypeScript + Vite + Element Plus + OpenLayers + ECharts
- **项目类型**：大型交通基础设施综合监管系统

## 项目简介

舟山跨海大桥综合监管平台是为舟山跨海大桥设计的综合性智能监管系统，旨在提升大桥管理的智能化、信息化水平。平台集成了桥梁实时监控、预警管理、应急指挥、车流统计、视频监控、态势感知等核心功能，通过地图可视化、数据分析和智能预警，为大桥运营管理提供全方位的技术支撑。

## 技术栈

### 核心技术
- **前端框架**：Vue 3.5.13 + TypeScript 5.7.2
- **构建工具**：Vite 5.4.11
- **UI组件库**：Element Plus 2.8.8
- **状态管理**：Pinia 2.2.6
- **路由管理**：Vue Router 4.4.5

### 专业领域
- **地图可视化**：OpenLayers 10.4.0（海图配置、船舶打点、定位偏移修复）
- **数据可视化**：ECharts 5.5.0（流量统计、预警分析、热词提取）
- **视频播放**：EasyPlayerPro 5.1.4 + flv.js + rtsp2web（实时监控、历史回放）
- **富文本编辑**：@vueup/vue-quill 1.2.0

### 工具库
- **样式方案**：UnoCSS 0.64.1 + TailwindCSS 4.0.6 + Sass
- **HTTP请求**：Axios 1.7.8
- **国际化**：Vue I18n 10.0.5
- **图片处理**：html2canvas 1.4.1 + image-conversion 2.1.1 + vue-cropper 1.1.1
- **其他**：dayjs、lodash-es、crypto-js、js-cookie、nprogress、screenfull等

## 解决的技术难题

### 1. 视频窗口拉伸问题
**问题描述**：监控摄像头对应的MonitorPopup视频窗口比例失衡，视频加载后被拉伸变形。

**解决方案**：
- 使用Vue的:deep()选择器穿透scoped样式隔离，直接作用于EasyPlayerPro动态生成的video标签
- 强制设置视频容器比例为标准比例（16:9）
- 修复视频加载后被拉伸的恶性bug

### 2. 地图定位偏移问题
**问题描述**：地图定位不准确，点位偏移导致视觉效果不佳。

**解决方案**：
- 调整偏移参数，确保偏移位置在视觉中心
- 偏移量调整为700，适配不同分辨率屏幕
- 编写地图定位偏移功能说明文档和修复说明文档

### 3. 上传图片失败问题
**问题描述**：日报上传图片功能失败，影响用户体验。

**解决方案**：
- 检查并修复上传接口对接问题
- 完善图片兼容性处理，支持多种图片格式
- 添加错误提示，提升用户体验

### 4. 视频窗口拖拽异常
**问题描述**：警戒船监控窗口拖动异常，窗口尺寸巨大。

**解决方案**：
- 移除hasDragged标志位，确保每次拖拽都重新计算元素位置
- 添加可拖动组件，实现窗口拖动功能
- 限制窗口尺寸范围，防止窗口过大

### 5. 日报格式规范问题
**问题描述**：日报显示格式不符合公文规范，影响正式使用。

**解决方案**：
- 调整字体大小、行间距、段落间距，确保符合公文格式
- 使用html2canvas实现HTML转图片，设置2倍缩放确保高清输出
- 通过浏览器打印API实现PDF导出，兼容A4纸张、纵向打印
